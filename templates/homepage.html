Leukemia Detection And Classification Using CNN

About

Acute Leukemia is a disease in which infected White Blood Cells perform abnormally. There have two main categories of Leukemia which is Acute and chronic. Nithyaa, Kumar, Gokul & Aananthi (2021) stated that, a common type of Acute Leukemia is Acute lymphocytic Leukemia and Acute myeloid Leukemia, which develop faster and more rapidly, nd the adult population is highly affected by this type of cancer. Hence, automated detection and classification using image processing is introduced to examine the microscopic blood smears images and more accurately to detect Leukemia. Therefore this method has a great ability to learn highly abstracted feature of object and more efficient to identify object.
